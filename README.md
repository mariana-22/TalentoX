# TalentoX

# Descripción del Proyecto
Es una plataforma innovadora diseñada para evaluar habilidades reales de estudiantes, aprendices y técnicos mediante micro-pruebas inteligentes, retos prácticos y evidencias.
En vez de basarse en títulos o diplomas, la plataforma mide competencias demostradas, generando un Pasaporte Digital de Habilidades que se actualiza automáticamente con cada prueba.

# El Problema
En la educación y el empleo actuales:
- Se contrata más por hoja de vida que por habilidades reales.
- Muchos aprendices sí tienen capacidades, pero no tienen cómo demostrarlas.
- Los certificados tradicionales muestran horas, no niveles reales de habilidad.
- **TalentoX** busca resolver esta brecha.

# La Solución
**TalentoX** propone un sistema capaz de:

✔ Micro-pruebas inteligentes
Pruebas cortas, rápidas y adaptativas que miden precisión, tiempo y dificultad.

✔ Retos prácticos
Actividades que evalúan capacidades reales:
código, conexiones eléctricas, diagnósticos, análisis, redacción y más.

✔ Evidencias reales
Los usuarios pueden subir videos, capturas, archivos o proyectos.

✔ Certificaciones automáticas
El sistema calcula niveles (0–5) según desempeño y evidencias.

# ¿Cómo se evalúan las habilidades?
Cada prueba genera puntaje según estos parámetros:

1. Dificultad de la prueba
- Fácil → 1 punto
- Media → 3 puntos
- Difícil → 5 puntos

2. Tiempo de respuesta
- Rápido → +2
- Normal → +1
- Lento → 0

3. Consistencia

- Rendimiento estable durante varias pruebas → +1 extra
- Cada habilidad se actualiza según los puntos acumulados.

# Tipos de Pruebas y Retos
- Pregunta técnica ***(teórico-práctico)***
“Encuentra el error en este código y explica la solución.”

- Reto práctico
“Sube un video realizando una conexión eléctrica básica correctamente.”

- Prueba adaptativa
El sistema sube o baja la dificultad según los aciertos del usuario.

# Resultados del Usuario

- Cada usuario obtiene un Pasaporte de Habilidades con:
- Lista de habilidades
- Nivel (0–5)
- Última evaluación
- Evidencias
- Puntaje acumulado
- Historial de retos

Todo esto crea un perfil dinámico y auténtico de habilidades.
